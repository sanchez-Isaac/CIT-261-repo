JSON - Parse

The JSON.parse() method parses a JSON string, constructing the JavaScript value or object described by the string.

When receiving data from a web server, the data is always a string, you just have to convert it back to a usable object.

Example:

You can parse like this:

var txt = '{"name":"Isaac", "age":28, "city":"Mexico City", "height":"1.79","weight":"75Kg","major":"Software Engineering"}'

var obj = JSON.parse(txt);
document.getElementById("paragr").innerHTML = obj.city + ", " + obj.name + ", " + obj.age;

This will convert a string, into an JSON Object



Use the console and type " console.log(obj); " to see the object created by the use of JSON.parse and the original variable " console.log(txt); " .

JSON - Stringify

The JSON.stringify() method converts a JavaScript object or value to a JSON string
When sending data to a web server the data has to be a string.

Syntax:
JSON.stringify(obj, replacer, space)

Paramenter Description
obj Required. The value to convert to a string
replacer Optional. Either a function or an array used to transform the result. The replacer is called for each item.
space Optional. Either a String or a Number. A string to be used as white space (max 10 characters), or a Number, from 0 to 10, to indicate how many space characters to use as white space.

You can use the JSON.stringify function like this:

var obj = { "name":"Isaac", "age":28, "city":"Mexico City", "height":"1.79","weight":"75Kg","major":"Software Engineering"};

var myJSON = JSON.stringify(obj);

document.getElementById("demo").innerHTML = myJSON;

Remember:

The server needs to receive strings, not objects.
